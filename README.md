# 🏓 Pong with AI

Аркадная игра Pong с интеллектуальным противником на основе Q-learning (обучение с подкреплением) и иерархическим обучением.

## 🎮 О проекте

Классическая игра Pong, где вы играете против AI, который обучался с использованием reinforcement learning. Уникальность игры в **иерархической системе обучения**: каждый следующий уровень сложности обучался против предыдущего.

### ✨ Особенности
- 🎯 **Три уровня сложности** с иерархическим обучением
- 🤖 **AI противник** на Q-learning
- 🎨 **Красивая графика** и звуковые эффекты
- 🖱️ **Управление мышью**
- 📊 **Система подсчета очков**
- 🚀 **Автоматическая установка** — просто скачай и играй

## 🎯 Уровни 

### 🟢 ПЕРВЫЙ
Противник использует детерминированный алгоритм, который:
- Анализирует траекторию мяча
- Предсказывает точку касания
- Позиционируется оптимально для отбития

### 🟡 ВТОРОЙ
AI обучен с использованием Q-learning, где в качестве противника выступал алгоритм с легкого уровня. Модель научилась предугадывать и противодействовать базовому алгоритму.

### 🔴 ТРЕТИЙ
Наиболее продвинутый AI, обученный в играх против модели со среднего уровня. Усвоил сложные тактики и контр-тактики, максимально приближен к игре человека.

## 🚀 Быстрый старт

### Windows (проще некуда)
1. **Установите Python 3.6+** с [официального сайта](https://www.python.org/downloads/)
2. **Скачайте игру** (клонируйте репозиторий или скачайте ZIP)
3. **Запустите игру** — дважды кликните на файл **`run.bat`**

Всё! При первом запуске игра автоматически:
- ✅ Создаст изолированное виртуальное окружение
- ✅ Установит все необходимые библиотеки
- ✅ Скачает файлы моделей (по ~27 МБ) с Hugging Face
- ✅ Запустит игру

При следующих запусках игра будет стартовать мгновенно — просто снова кликните на `run.bat`!

## 🗑️ Удаление
Чтобы полностью удалить игру, просто удалите папку с ней.

📥 Установка вручную (для разработчиков)
Если хотите установить всё вручную:

bash
# Клонируйте репозиторий
git clone https://github.com/nx-pixel/pong.git
cd ping_pong_with_AI

# Создайте виртуальное окружение
python -m venv pong_ai_env

# Активируйте его
# Windows:
pong_ai_env\Scripts\activate
# Linux/Mac:
source pong_ai_env/bin/activate

# Установите зависимости
pip install -r requirements.txt

# Скачайте модели
python download_models.py

# Запустите игру
python ping_pong_with_AI.py

❓ Часто задаваемые вопросы
❔ Нужно ли платить?
Нет, проект полностью бесплатный!

❔ Почему при первом запуске долго загружается?
Скачиваются файлы моделей (~27 МБ) с Hugging Face. При следующих запусках загрузка не требуется.

❔ Можно ли играть без интернета?
Да! После первого запуска модели сохраняются локально, интернет больше не нужен.

❔ Как выбрать уровень сложности?
При запуске игры появится меню с выбором уровня.

❔ Не устанавливается pygame / numpy
Убедитесь, что Python установлен правильно и добавлен в PATH. При проблемах запустите run.bat от имени администратора.
